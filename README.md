# BMW Showcase Store

A premium car rental website showcasing BMW models with an interactive shopping cart and responsive design.

## Features

- **Car Showcase**: Interactive slider displaying BMW models (3 Series, 5 Series, 7 Series, X5, i8).
- **Model Filtering**: Filter cars by category (S1, X2, X5).
- **Shopping Cart**: Add/remove cars, persistent storage using localStorage.
- **Responsive Design**: Mobile-friendly layout with Tailwind CSS.
- **Navigation**: Sticky header with mobile menu toggle.

## Technologies Used

- **HTML5**: Structure and content.
- **CSS3**: Styling with Tailwind CSS framework.
- **JavaScript**: Interactive features (slider, cart, filtering).

## Setup

1. Clone or download the project files.
2. Open `index.html` in a web browser.
3. No additional setup required (static website).

## Usage

- Navigate through pages: Home, Models, About, Order.
- On Models page: Select category to filter cars, click "Add to Cart".
- Click basket icon to view/manage cart.
- Cart data persists across sessions.
